# Excel article screenshots

Drop screenshots for the Excel learn track (`/learn/excel/...`) into this folder.

## How it works
Each article has one or more screenshot slots. Until a file exists, the slot shows a
dashed placeholder that displays the **exact filename it expects** (in green). Add a PNG
with that name here and it appears automatically on the page. No code change needed.

## Rules
- Format: **PNG** (JPG works too, but keep the `.png` name the slot expects).
- Width: roughly **1000-1400px**, cropped to the relevant part of Excel.
- Filename must match exactly (case-sensitive), e.g. `ex1-t1-interface.png`.

## Currently expected files
- `ex1-t1-interface.png`  — the Excel window with ribbon / name box / formula bar / grid
- `ex1-t2-sheets.png`     — the worksheet tabs at the bottom
- `ex1-t3-range.png`      — a selected range A1:C10 in the grid
- `ex2-t2-paste-special.png` — the Paste Special dialog
- `ex2-t3-fill-handle.png`   — dragging the fill handle to extend a series

More filenames will be listed here as new topics are added. You can also just open any
article on the site: every empty slot shows the filename to use.

## Adding files

- Upload directly on GitHub: repo -> `excel-img` -> Add file -> Upload files.
